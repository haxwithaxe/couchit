Couchit allow you to easily build and maintain your website.

Add your content and get started right away.

Couchit is still a beta release, but you already have:

    * a simple and intuitive user interface
    * the possibility to create and edit pages
    * the possibility to choose your site URI
    * an easy way to choose your website color scheme
    * a nice integration with Friendpaste
    * ...
